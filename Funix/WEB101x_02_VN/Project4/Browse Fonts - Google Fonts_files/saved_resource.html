Use on the web

To embed a font, copy the code into the <head> of your html

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@700&family=Roboto:wght@300&family=Rowdies:wght@700&display=swap" rel="stylesheet">

CSS rules to specify families

  1. font-family: 'Noto Sans', sans-serif;
  1. font-family: 'Roboto', sans-serif;
  1. font-family: 'Rowdies', cursive;
